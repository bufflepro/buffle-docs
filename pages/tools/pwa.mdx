# PWA (Progressive Web App) `Coming Soon`

**Buffle Pro as a PWA!** We are thrilled to announce that Buffle Pro will also be available as a Progressive Web App (PWA). This will allow users to access all the functionalities of Buffle Pro as an app on their PC or Mac, providing a seamless experience without the need for installation. Enjoy the flexibility of using Buffle Pro in a browser while benefiting from app-like performance and features!
